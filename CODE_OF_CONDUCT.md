# Code Of Conduct

This project aims to be a practical, respectful place to build developer tools.

## Expected Behavior

- Be direct, constructive, and specific.
- Assume good intent, but correct mistakes with evidence.
- Keep discussion focused on the work.
- Respect different experience levels and operating systems.
- Accept maintainer decisions about scope, safety, and project direction.

## Unacceptable Behavior

- Harassment, threats, insults, or discriminatory language.
- Sexualized language or imagery.
- Publishing private information without permission.
- Repeatedly derailing technical discussions.
- Abusing issues, pull requests, or reviews for spam or personal attacks.

## Enforcement

Maintainers may edit, hide, or delete comments; close issues or pull requests;
or block participants when behavior harms the project.

To report a conduct issue, contact the repository owner through GitHub. If the
issue is security-sensitive, follow [SECURITY.md](SECURITY.md) instead of
posting details publicly.
