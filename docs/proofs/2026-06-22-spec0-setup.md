pre-existing untracked `tickets/` folder from ticket draft creation.

`rg --files` confirmed the repo shape before setup: Rust source under `src/`,
CLI integration tests under `tests/`, CI under `.github/workflows/`, Remotion
demo files under `demo/`, and root ticket drafts under `tickets/`.

`git diff --check` passed with no whitespace errors. Because the new `docs/`
files are untracked, this command does not inspect their content until they are
staged or added with intent-to-add.
