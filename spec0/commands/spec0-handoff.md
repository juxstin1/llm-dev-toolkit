# Spec0 Handoff

Create a Spec0 handoff:

`$ARGUMENTS`

## Include

- Goal and current status.
- Files changed and why.
- Commands or checks run, with pass/fail status.
- Decisions future agents must preserve.
- Open risks, blockers, and next actions.
- Any user-specific instructions or repo-local agent instructions that mattered.

Keep the handoff factual and short enough for the next agent to read before acting.
